# Development Setup

## Prerequisites
- Flutter SDK
- Android Studio or Android SDK tools
- VS Code with Flutter and Dart extensions

## Local Setup
1. Clone repository
2. Run `flutter pub get`
3. Configure environment variables (Supabase URL/key)
4. Run `flutter run`

## Supabase
Create your own Supabase project for local development and set:
- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`

Never commit production credentials.

## Useful Commands
- `flutter doctor`
- `flutter pub get`
- `flutter analyze`
- `flutter test`
- `flutter run`
